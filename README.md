# AI-Based Autonomous Navigation System

## Project Overview

This project demonstrates an AI-Based Autonomous Navigation System using Python and A* path planning in a virtual simulation environment.

The robot autonomously navigates from a start position to a destination while avoiding obstacles.

---

## Features

- Autonomous navigation
- Obstacle avoidance
- A* path planning algorithm
- Virtual simulation environment
- Real-time path visualization

---

## Tech Stack

- Python
- NumPy
- Matplotlib
- Google Colab

---

## Simulation Workflow

1. Create virtual environment
2. Add obstacles
3. Apply A* path planning
4. Generate shortest path
5. Simulate robot movement
6. Visualize navigation
